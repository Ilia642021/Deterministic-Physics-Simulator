# Deterministic Physics Simulator

## О проекте
Данный проект представляет собой демонстрационную модель **детерминированной двумерной физической симуляции**. Основная цель — создать систему, которая при идентичных начальных условиях всегда выдает строго одинаковый результат, независимо от количества прошедших итераций (тиков) или платформы запуска.

Проект является практической частью первой статьи из трилогии **"Мат. казино"**, посвященной исследованию природы визуальной случайности в интерактивных системах.

## Ключевые идеи
- **Детерминизм**: Полная воспроизводимость состояния системы. Если `seed` и начальные параметры объектов совпадают, результат симуляции будет идентичен до последнего бита.
- **Разделение вычислений и визуализации**: Физический движок рассчитывает состояние системы независимо от рендеринга. Результаты сохраняются в структурированный JSON-файл, который затем может быть воспроизведен визуализатором.
- **Continuous Collision Detection (CCD)**: Использование алгоритмов непрерывного обнаружения столкновений позволяет избежать "просачивания" объектов сквозь препятствия даже при высоких скоростях или на границах временных шагов.

## Суть эксперимента
В рамках статьи "Мат. казино" данный симулятор демонстрирует, что внешне хаотичное и непредсказуемое поведение (множество шаров в сложном лабиринте с тысячами столкновений) может быть полностью предопределенным. Это ставит вопрос о том, что именно мы воспринимаем как "случайность" в цифровых средах: реальный стохастический процесс или просто сложную детерминированную динамику.

## Технологический стек
- **Язык**: Python 3
- **Вычисления**: NumPy (векторизация, линейная алгебра)
- **Визуализация**: Matplotlib
- **Формат данных**: JSON (схема для хранения кадров и событий)

## Структура проекта
- `physics_engine.py`: Ядро симулятора, расчет столкновений и состояний.
- `visualizer.py`: Универсальный рендерер для отображения результатов симуляции.
- `collision_analyzer.py`: Инструмент для анализа плотности столкновений в кадрах.
- `main.py` / `main2.py`: Примеры настройки и запуска симуляций.

## Установка и запуск
1. Установите зависимости:
   ```bash
   pip install numpy matplotlib
   ```
2. Запустите демонстрационную симуляцию:
   ```bash
   python main2.py
   ```
